# RUC Survival Guide / 人大学生生存指南

> 非官方、学生维护的中国人民大学学习与生存指南。

本项目希望把人大同学分散在各处的课程经验、课程资料、政策链接、升学与就业路径经验集中整理成一个可长期维护的网站。它既是一本可以阅读的指南，也是一套可以持续扩展的课程资料库。

## 项目定位

- 面向对象：以本科生为主，兼容部分硕博课程资料。
- 内容范围：课程指南、课程资料、保研、出国、就业、科研入门、转专业、辅修、校园生活索引。
- 组织方式：个人文章 + 社区资料库混合。主线文章由维护者主导，课程资料和课程体验逐步开放给社区贡献。
- 项目身份：本项目不是中国人民大学官方项目，所有观点和资料仅供参考。

## 首版结构

```text
docs/                  # 网站正文
  guide/               # 入门与学习规划
  courses/             # 课程入口：首字母索引 + 学院入口
  survival/            # 保研、出国、就业、科研等决策指南
  life/                # 校园生活轻量索引
  resources/           # 政策、链接、工具资源
  contribute/          # 投稿、版权、下架、模板
materials/             # 课程资料归档
  courses/             # 按学院 + 课程归档
overrides/             # MkDocs Material 自定义模板
```

## 课程双入口

1. **课程名首字母排序**：适合直接查找某门课。
2. **学院 + 课程**：适合按开课单位浏览和维护资料。

课程类型、学分、适用专业、是否通识课等信息放入课程页元数据和正文标签中，不作为首版主导航。

## 本地预览

```bash
pip install -r requirements.txt
mkdocs serve
```

默认访问地址通常是 `http://127.0.0.1:8000`。

## 部署

仓库包含 GitHub Pages 工作流。推送到 GitHub 后，可在仓库设置中启用 Pages，并选择 GitHub Actions 作为部署来源。
