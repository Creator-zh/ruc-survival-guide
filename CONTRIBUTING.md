# 贡献指南

欢迎补充课程资料、课程体验、政策入口和个人经验文章。

## 贡献类型

- 新增或修改课程页。
- 新增课程资料或资料链接。
- 更正过期信息。
- 撰写个人经验文章。
- 补充保研、出国、就业、科研、转专业等路径经验。

## 基本要求

1. 标注信息来源和适用时间。
2. 涉及课程体验时，尽量写清楚学期、任课教师、考核方式和个人背景。
3. 不提交他人隐私、身份证件、成绩单、联系方式等敏感信息。
4. 不进行人身攻击。
5. 对第三方资料标注来源和权利归属。

## 文件命名

课程正文：

```text
docs/courses/schools/学院名/课程名.md
```

课程资料：

```text
materials/courses/学院名/课程名/
```

建议资料文件名包含年份、学期和类型：

```text
2025-fall-midterm-memory.md
2024-spring-final.pdf
2025-fall-notes-author.md
```

## 大文件

PDF、PPT、压缩包等大文件建议使用 Git LFS。特别大的资料可以放到 GitHub Release 或外部存储，并在课程页中放链接。

## 评论

课程页面下方评论区使用 GitHub Discussions + giscus。评论由 GitHub 账号署名发布，代表评论者个人观点。
