# 课程指南

课程指南是本项目的核心资料库。

## 两个入口

1. [按课程名首字母排序](alphabetical/index.md)：适合直接搜索某门课。
2. [按学院 + 课程](schools/index.md)：适合按开课单位浏览课程和维护资料。

## 每门课应该包含

- 课程基本信息。
- 学习经验和适合人群。
- 课程大纲、参考书、参考资料。
- 作业、期中、期末、真题或回忆版资料。
- 任课教师、考核方式、课程体验。
- 资料目录和版权说明。
- GitHub Discussions 评论区。

## 资料存放

网站正文放在 `docs/courses/`，课程资料放在 `materials/courses/`。大文件建议使用 Git LFS 或 GitHub Release 管理。
