# 绩点与学习策略

绩点是重要信号，但不是完整能力本身。不同路径对绩点的要求不同：

- 保研：绩点和排名通常非常关键，同时还要看科研、竞赛、英语、综合评价等。
- 出国：绩点重要，但项目匹配、科研、推荐信、语言成绩和文书同样重要。
- 就业：绩点可能影响简历筛选，但实习、项目、技能和面试表现更直接。

## 学习策略

- 尽早识别课程考核方式：闭卷、开卷、论文、展示、项目、平时作业。
- 不同课程使用不同策略：数学课重练习，论文课重阅读和结构，项目课重过程管理。
- 保留复习材料：往年题、错题、笔记、作业答案和课程重点。

## 不建议

- 只根据“给分好不好”选课。
- 把所有时间都压在短期绩点上，完全忽略长期能力。
- 不了解保研、出国、就业规则就盲目做规划。
