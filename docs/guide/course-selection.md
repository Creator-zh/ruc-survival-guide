# 选课指南

选课不是单纯抢课，而是在毕业要求、兴趣、绩点、时间表和长期路径之间做平衡。

## 选课前先看什么

- 培养方案：确认必修、限选、任选、通识、实践等要求。
- 课程大纲：确认先修要求、作业量、考核方式。
- 历史体验：确认课程难度、给分、点名、课堂风格和考试风格。
- 自己的本学期目标：保绩点、补基础、探索方向、准备实习或科研。

## 课程页面应提供的信息

- 任课教师和开课学期。
- 学分、课程类别和适用专业。
- 考核方式和工作量。
- 课程资料、参考书、真题或回忆版题目。
- 同学的课程体验和评论区讨论。

## 风险提示

课程评价容易过期。阅读任何评价时都要看清楚学期、任课教师和投稿者背景。
