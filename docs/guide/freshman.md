# 新生入门

这篇文章用于回答“刚入学之后应该先搞清楚什么”。

## 开学前后优先级

1. 确认培养方案、毕业要求和学分结构。
2. 熟悉选课系统、教务系统、邮箱、校园网、图书馆和常用通知入口。
3. 建立自己的资料管理方式：课程文件夹、笔记系统、待办清单。
4. 先观察课程和学院节奏，不要急着把每条路径都定死。

## 新生最容易忽略的事

- 官方通知入口比群聊转发更可靠。
- 课程体验有明显年份差异，同一门课不同老师差别可能很大。
- 绩点重要，但不是唯一目标；如果计划保研、出国或就业，评价体系会不同。
- 越早整理课程资料，后期复习和分享成本越低。

## 建议补充

- 新生常用链接。
- 各学院培养方案入口。
- 选课、体测、英语分级、军训等高频事项。
