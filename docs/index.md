# 人大学生生存指南

这里是一份非官方、学生维护的中国人民大学学习与生存指南。

项目目标不是替代学校通知，也不是给出唯一正确答案，而是把同学们真正需要的经验、资料、路径判断和链接集中起来，减少重复踩坑。

## 你可以从这里开始

- 如果你是新生，先看[新生入门](guide/freshman.md)。
- 如果你要选课，先看[选课指南](guide/course-selection.md)。
- 如果你在找课程资料，进入[课程指南](courses/index.md)。
- 如果你在考虑保研、出国、就业或科研，进入[生存参考](survival/index.md)。
- 如果你想投稿或补充资料，阅读[参与贡献](contribute/index.md)。

## 内容边界

本项目以本科生为主，也收录部分硕博课程资料。课程页面可以讨论任课教师、考核方式、课程体验和学习建议，但请尽量使用可核验、事实化、标注时间的表述。

具体导师评价、实验室评价和个人隐私内容不属于本项目首版范围。

## 重要声明

本项目不是中国人民大学官方项目。政策信息请以学校、学院和相关部门的最新通知为准。课程资料的知识产权归原作者、任课教师、课程团队、学校或其他权利人所有。
