# 就业

就业准备应尽早和课程、项目、实习结合，而不是只在秋招前开始。

## 常见路径

- 企业就业。
- 公务员和选调。
- 事业单位。
- 金融、咨询、互联网、快消、教育等行业。
- 继续读研后就业。

## 准备框架

| 阶段 | 建议 |
| --- | --- |
| 大一 | 探索行业和技能，不急着定死方向 |
| 大二 | 做第一段项目、社团、实习或科研尝试 |
| 大三 | 形成简历主线，争取核心实习 |
| 大四 | 秋招、考公、考研或留学申请并行管理 |

## 个人文章方向

- 如何从零写第一版简历。
- 人大不同专业的常见就业方向。
- 实习、绩点、学生工作之间如何取舍。
- 秋招信息源和准备节奏。
