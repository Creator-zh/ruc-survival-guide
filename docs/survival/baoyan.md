# 保研

保研信息高度依赖年份、学院和具体政策。本文应作为“准备框架”，具体规则以学校和学院当年通知为准。

## 需要关注的变量

- 专业排名和绩点。
- 综合评价规则。
- 科研、竞赛、论文、社会实践等加分项。
- 英语成绩。
- 夏令营、预推免、九推时间线。

## 建议时间线

| 时间 | 重点 |
| --- | --- |
| 大一 | 稳住基础课和绩点，了解本专业培养方案 |
| 大二 | 确认是否走保研路径，尝试科研或竞赛 |
| 大三上 | 提升核心课程成绩，准备英语和科研材料 |
| 大三下 | 准备夏令营材料，整理简历、成绩单、推荐信 |
| 大四上 | 关注预推免和九推流程 |

## 个人文章方向

- 保研是否值得卷。
- 低年级如何判断自己有没有保研可能。
- 科研、竞赛、绩点之间如何取舍。
- 本校保研和外校保研的准备差异。
