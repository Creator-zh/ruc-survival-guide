# 出国

出国申请不是单一考试，而是长期组合：绩点、语言、科研/实习、推荐信、文书、项目匹配和资金规划。

## 常见方向

- 授课型硕士。
- 研究型硕士。
- 博士申请。
- 交换、暑研和短期项目。

## 准备要素

| 要素 | 说明 |
| --- | --- |
| GPA | 重要基础，越早稳定越好 |
| 语言 | TOEFL / IELTS 等 |
| 标化 | GRE / GMAT 等，视项目要求 |
| 推荐信 | 需要真实互动和可信评价 |
| 经历 | 科研、实习、项目、论文、竞赛 |
| 文书 | 解释动机、匹配度和能力证据 |

## 个人文章方向

- 如何判断自己适不适合出国。
- 授课型硕士和研究型项目的差别。
- 什么时候开始准备语言和科研。
- 文科、社科、商科、理工科申请差异。
