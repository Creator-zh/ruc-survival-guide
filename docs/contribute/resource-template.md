# 资料投稿模板

提交课程资料时，请尽量提供以下信息。

```markdown
## 资料基本信息

- 课程名称：
- 开课学院：
- 任课教师：
- 学期：
- 资料类型：课程大纲 / 课件 / 笔记 / 作业 / 真题 / 回忆版 / 参考书 / 链接
- 来源：
- 是否包含个人信息：
- 是否需要匿名：

## 资料说明

请说明资料适用范围、使用建议、是否完整、是否有明显错误。

## 权利声明

我理解该资料可能涉及原作者、任课教师、课程团队或其他权利人的知识产权。若权利人要求删除或修改，维护者可以按下架流程处理。
```

## 推荐目录

```text
materials/courses/学院名/课程名/
  README.md
  syllabus/
  exams/
  notes/
  slides/
  links.md
```
