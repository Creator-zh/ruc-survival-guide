# 参与贡献

本项目希望长期由学生社区共同维护，但首版仍由维护者控制整体结构和质量。

## 你可以贡献什么

- 课程资料：大纲、笔记、真题、回忆版、参考书、链接。
- 课程经验：学习方法、作业节奏、考试风格、任课教师差异。
- 生存参考：保研、出国、就业、科研、转专业、辅修等个人文章。
- 校园生活：高频办事流程和官方入口。

## 贡献入口

- 修改正文：提交 Pull Request。
- 提交资料：提交 Pull Request 或 Issue。
- 课程体验：直接在课程页下方用 GitHub 账号评论。
- 下架或更正：使用仓库 Issue 模板。

## 内容分层

- **正文文章**：维护者审核后合并。
- **课程资料**：尽量标注来源和时间，大文件单独管理。
- **课程评论**：不预审核，但出现隐私、攻击、明显虚假内容时可处理。
