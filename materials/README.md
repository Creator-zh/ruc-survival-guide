# 资料库

这里存放课程资料、链接和归档说明。

资料库按“学院 + 课程”组织：

```text
materials/courses/学院名/课程名/
```

每门课建议包含：

```text
README.md
syllabus/
exams/
notes/
slides/
links.md
```

## 资料命名

推荐包含年份、学期、类型：

```text
2025-fall-final.pdf
2025-fall-midterm-memory.md
2024-spring-notes-author.md
```
